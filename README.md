# Mato-Ai

Mato is a new generation of AI-powered development models designed to go beyond simple code completion. Instead of just suggesting snippets, Mato models actively build, manage, and run full projects inside a live workspace environment.

⚡ Available Models

* Mato Default – Fast, general-purpose assistant for everyday coding tasks
* Mato Code (480B) – High-capacity model optimized for complex code generation and reasoning
* Mato Dev – Advanced development agent with deeper understanding, planning, and execution capabilities

⸻

🚀 What Makes Mato Different?

Unlike traditional AI tools, Mato operates inside a fully interactive workspace, meaning it can directly interact with your project files, environment, and runtime.

📁 Workspace Capabilities

list_files
read_file
write_file
create_folder
delete_path
move_path
copy_path
search_files_by_name
search_files_by_text
get_file_metadata

💻 Terminal Access

run_command
install_package

⚙️ Runtime Control

start_server
stop_server
restart_server
set_env_var

This allows Mato to:

* Build full applications from scratch
* Edit and refactor entire codebases
* Install dependencies and run commands
* Launch and manage live servers
* Debug and fix issues in real time
